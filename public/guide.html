📋 모두의 학급 — 사용 안내
인왕중학교 안경희 선생님 · 학급 전자칠판 운영 가이드
작성일: 2026년 6월 29일
대상: 담임 교사
① 목적

▸ 공지사항·일정·칭찬 도장 등 학급 운영 정보를 실시간으로 학생과 공유

▸ 기존 분필 칠판과 달리, TV·프로젝터 등 디스플레이에 연결하여 전자칠판 기능을 최대한 활용

▸ 공지·타이머·발표자·선거 등 학급 운영에 필요한 기능을 한 곳에 통합하여 편의성 향상

② 사용 방법
STEP 1 · 초기 접속 및 비밀번호 설정
  • 브라우저에서 서비스 주소 접속 → 모두의 학급 메인 화면 확인
  • 교사 패널 버튼 클릭 (최초 접속 시 비밀번호 없이 입장)
  • 교사 패널 하단 🔐 교사 모드 비밀번호 항목에서 비밀번호 설정
  • 설정 후 전자칠판 화면의 우하단 자물쇠 아이콘 클릭 시 비밀번호 확인
  • ⚠ 비밀번호는 이 기기의 브라우저에만 저장됩니다 (다른 기기에서는 재설정 필요)
STEP 2 · 전자칠판 표시
  • 메인 화면 → 전자칠판 보기 클릭
  • TV·프로젝터 연결 후 F11 키로 전체화면 전환 권장
  • 공지사항·D-Day·TOP3 도장·교육자료·학급 사진이 자동 표시됨
STEP 3 · 교사 패널에서 내용 관리
  • 전자칠판 우하단 자물쇠 클릭 → 비밀번호 입력 → 교사 패널 이동
  • 또는 메인 화면 → 교사 패널 버튼으로 직접 접속
③ 주요 기능
📢 공지사항
긴급·일반·제출·시간표 유형별 공지 작성. 만료일 설정 가능.
📅 D-Day 카운트다운
시험·행사·방학 등 날짜 등록 시 자동으로 남은 일수 표시.
🐥 칭찬 도장
학생별 도장 부여, 주간·누적 현황 집계, 상위 3명 자동 표시.
⏱️ 집중 타이머
수업 중 전자칠판에 큰 화면으로 타이머 표시.
🎲 랜덤 발표자
학생 명단에서 무작위로 발표자를 선정하여 화면에 표시.
🗳️ 학급 선거
투표 항목 생성 후 학생들이 직접 투표, 실시간 결과 집계.
🧹 청소 당번
청소 구역별 담당 학생 배정 및 관리.
📸 학급 사진
사진 업로드 시 전자칠판에서 슬라이드쇼로 자동 표시.
📖 오늘의 교육
교육자료 카드 작성 후 전자칠판에 순환 표시.
④ 주의사항
  • 비밀번호는 이 기기의 브라우저에만 저장됩니다. 다른 기기에서 교사 패널 접속 시 새로 설정해야 합니다.
  • 학급 사진은 서버에 업로드되지 않고 이 기기에만 저장됩니다. 기기가 바뀌면 재업로드가 필요합니다.
  • 공지사항·학생 정보·도장 현황은 Firebase를 통해 여러 기기 간 자동으로 동기화됩니다.
  • 공용 PC 사용 시 수업 종료 후 브라우저 로컬 데이터 삭제(설정 → 방문 기록 삭제)를 권장합니다.
  • Chrome 브라우저 + 전체화면(F11) 환경에서 가장 안정적으로 작동합니다.
  • 학생 이름은 화면 표시 시 앞·뒤 글자만 표시되고 가운데는 ✦로 마스킹 처리됩니다.
